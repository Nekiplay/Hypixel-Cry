# Features
### Commands:
- /cry - Open settings menu
- /path - Set pathfinder

### Macros:
- Auto right click (Chests, Leavers, Skulls)
- Auto close chests in dungeons

### ESP:
- Dark Monolith in Dwarden Mines
- Glowing Mushroom
- Treasure Hunter
